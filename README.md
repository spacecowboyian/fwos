fwos
====

Four Words On Shirts - UI

## Requirements

- Node.js 18+
- npm

## Install

```bash
npm install
```

## Run locally (Next.js)

```bash
npm run dev
```

App runs at `http://localhost:3000` by default.

## Production

```bash
npm run build
npm run start
```

## Project structure note

- This project now runs fully on Next.js (`pages/`, `public/`, and root-level `styles/`).
- Legacy Grunt/Bower source files were removed as part of the cleanup.
